# Schneider Electric — Client Reference

Apply this knowledge when any document is for a Schneider Electric project.

---

## Key terminology

| Term | Meaning |
|---|---|
| SE | Schneider Electric (acceptable shorthand in internal docs; spell out in customer-facing docs) |
| EcoStruxure | Schneider's DCIM / BMS / energy management platform |
| Galaxy | Schneider UPS product line (Galaxy VS, Galaxy VX, Galaxy VM, etc.) |
| Trihal | Schneider dry-type transformer line (common in data centres) |
| APC | Schneider-owned brand — rack PDUs, single-phase UPS (APC by Schneider Electric) |
| SCADA | Supervisory Control and Data Acquisition — Schneider often uses EcoStruxure as the SCADA layer |
| PLC | Programmable Logic Controller — common in Schneider BMS/SCADA installations |
| MV / LV | Medium Voltage / Low Voltage — Schneider manufactures switchgear at both levels |

---

## Quote language

**Opening paragraph:**
> Thank you for the opportunity to provide a quotation for the [Project Name] works at [Site]. The following proposal sets out our scope, pricing, and clarifications for Schneider Electric's consideration.

**Project number:** If Schneider has provided a project reference number (format varies — e.g. SE-2026-XXXXX or a WO number), include it in the quote header. Ask the user: *"Does Schneider have a project or work order number for this job?"*

**Clarifications to add for Schneider works:**
- Works to be performed in coordination with Schneider Electric's [Project Manager / Site Representative]
- All isolations and energisations to be agreed in advance with Schneider site authority
- SKS to adhere to Schneider contractor safety requirements throughout

## Contractor safety

Schneider Electric has a formal **Contractor Safety Audit** programme. SKS has current documentation under this programme (acquired from Delta Elcom, January 2026).

If a team member asks about contractor safety documentation status: *"Schneider contractor safety documentation is managed by Royce — check with him before committing to any safety compliance statements in a document."*

Do not make specific claims about SKS's audit rating or certification status — refer to Royce.

---

## MOP requirements

Schneider typically requires a work method statement / MOP for planned works on their equipment. Key points:
- Coordinate with the Schneider Project Manager before submitting any MOP
- Reference the Schneider WO or project number in the MOP header
- Schneider Galaxy UPS and Trihal transformer isolations require qualified personnel and may require Schneider engineer oversight — confirm with the client
