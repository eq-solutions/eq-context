# Session 2026-06-09 — S2-9 Per-Consumer Key Isolation: Receiver Updates

## What was built

S2-9 per-consumer key isolation landed on eq-shell (`claude/s2-9-per-consumer-keys`). This session updated all three consuming apps to read the new dedicated keys with fallback to `EQ_SECRET_SALT`.

### Changes made

**eq-solves-field — `netlify/functions/verify-pin.js`**
- Added `FIELD_HANDOFF_KEY = process.env.EQ_FIELD_HANDOFF_KEY || process.env.EQ_SECRET_SALT`
- `verifyShellToken()` now uses `FIELD_HANDOFF_KEY` instead of `SECRET_SALT`
- Global `SECRET_SALT` untouched — still backs session tokens and all other HMAC ops

**eq-cards — `netlify/functions/shell-verify.js`**
- Shell session cookie (signed with `EQ_SESSION_SALT` under S2-9) now verified with `EQ_SESSION_SALT || EQ_SECRET_SALT`
- Cards was the only consumer verifying the session cookie without the new key

**eq-solves-service — `app/api/shell-auth/route.ts`**
- Legacy service token path updated to `EQ_SERVICE_HANDOFF_KEY ?? EQ_SECRET_SALT`
- Bridge path (`EQ_SHELL_BRIDGE_SECRET`) and Supabase JWT path untouched — already isolated
- Phase 3 (Supabase JWT fast path) confirmed already implemented and live in shadow mode

## What was assessed but not actioned

- **EQ_SECRET_SALT rotation** — not urgent (exposed value was demo salt from April chat, not prod). Deferred to a planned maintenance window once S2-9 merges and new env vars are set in Netlify.
- **JTI on ShellTokens** — deferred. No-op without a revocation store; not worth the token format churn yet.
- **Shadow mode flip (token-exchange Phase 2 → prod)** — config change (`shell_control.platform_config`), not code. Hold until end-to-end smoke test.
- **Legacy service token removal** — the TODO in shell-auth/route.ts to remove the legacy path. Hold until EQ_SHELL_BRIDGE_SECRET is confirmed set on both Shell and Service Netlify envs.

## Key finding

Service's shell-auth/route.ts already has full Phase 3 (Supabase JWT) support. Cards doesn't sign tokens — it only verifies the Shell session cookie. No new token surface to isolate; just key alignment.
