# Session 2026-07-02 (part 7) — eq-cards OCR spinner freeze root-caused + fixed, demo account cleanup

New `_compressForWeb` mirrors the existing resize/JPEG logic but swaps `canvas.toDataURL()` + base64 round-trip for `canvas.toBlob()` + `Blob.arrayBuffer()` (encodes off the main thread). Native iOS/Android untouched — still calls the existing plugin. Added `web: ^1.1.0` as a direct pubspec dependency (was transitive-only via the plugin).
